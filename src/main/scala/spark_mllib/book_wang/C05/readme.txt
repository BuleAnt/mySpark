协同过滤算法是最常用的算法,主要有两种具体形式:
基于用户的推荐算法和基于物品的推荐算法.
ALS(alternating least squares)是交替最小二乘法的简称,也是MLlib的基础推荐算法

协同过滤
协同过滤算法又称为"集体算法"方法,基本思想是利用任性的相似性进行相似比较.
协同过滤(Collaborative Filtering)是基于军体用户或物品的典型推荐算法.
主要有两种:
	1,通过考察具有相同爱好的用户对相同物品的评分标准进行计算
	2,考察具有相同特性的物品从二推荐给选择了某件物品的用户

协同过滤的不足:
	在实际应用中,基于用户和物品的推荐算法军事最常用的协同过滤推荐算法,但是仍具不足之处:
	首先是基于用户的推荐算法,针对某些热点物品的处理不够准确,对于一些常用的物品推荐,
	其计算结果往往排在推荐的首位,而这样的推荐却没有实际应用意义.
	同事基于用户的推荐算法,旺旺数据量较为庞大,计算费事,由于热点的存在准确度也恨成问题.

	基于物品的推荐算法相对于用户的推荐算法,其数据量小很多,可以较为容易的生成推荐值,但其存在推荐同样(同类)物品的问题.
	例如,用户购买了某件物品,那么推荐系统可能会继续推荐相同的商品给用户,
	用户在购买意见商品后绝对不会在后买同类的商品,这样的推荐是完全是失败的.

欧氏距离相似度矩阵计算.
	d(x, y) = sqrt(∑((x(i)-y(i)) * (x(i)-y(i))))
	sim(x, y) = n / (1 + d(x, y))
余弦相似度矩阵计算.
	T(x,y) = ∑x(i)y(i) / sqrt(∑(x(i)*x(i)) * ∑(y(i)*y(i)))



欧氏距离相似度与余弦相似度比较:
欧氏距离相似度是以目标绝对距离作为衡量的标准,而余弦相似度是以目标差异的大小作为衡量标准.
一般欧几里得相似度用来表现不同目标的绝对差异性,分析目标之间的想速度与差异情况.
而余弦相似度更多的是对目标从方向趋势上区分,对特定坐标数字不敏感.




交替最小二乘法(ALS算法)
LS算法是数学优化技术,也是机器学习常用算法,通过最小话误差的平方和寻找数据的最佳函数匹配.
利用最小二乘法可以简单地球的位置的数据,并使得这些求得的数据与实际数据之间误差的平法和最小.
